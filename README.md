# **📊 Amazon Sales Analysis Dashboard (Power BI)**



### **Overview**

This project presents an interactive Amazon Sales Analysis Dashboard built using Microsoft Power BI. The dashboard provides insights into sales performance, customer behavior, product categories, payment methods, and delivery status through interactive visualizations and KPIs.



### 

### **Dashboard Preview**



!\[Amazon Sales Dashboard](Dashboard Screenshot.png)



### 

### **Key Features**



#### **KPI Cards**



* Total Sales
* Total Orders
* Average Order Value
* Delivered Orders
* Return Rate



#### **Interactive Filters**



* Order Date
* Product Category
* Customer City
* Payment Method
* Delivery Status



#### **Visualizations**



* Monthly Sales Trend
* Sales by Category
* Sales by City
* Top 10 Products
* Payment Method Distribution
* Delivery Status Analysis
* Quantity Sold by Category
